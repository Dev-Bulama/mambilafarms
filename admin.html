Your inbox is the registry
For investor privacy, registration data is never stored in the browser. Every submission fires an instant email to your admin address(es) containing the full investor record. Your email inbox is your searchable, permanent database.
1. Each new registration arrives as a styled email with every field — name, BVN, tier, phone, address, and more.
2. Use your email client's search and labels/folders to organise by tier, date, or investor type.
3. To build a spreadsheet registry, forward all registration emails to a Google Sheet using Google Apps Script, or manually paste the data — the email contains every field in a clean table format.
4. For a fully automated registry (live table + CSV export), add a single PHP file to your server and this dashboard can be upgraded with no other backend changes needed.
Why not localStorage? Browser storage is readable by any visitor via DevTools → Application. Sensitive investor data (BVN, TIN, address) must never sit there.
